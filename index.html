Six-Layer Moderation Architecture

Every customer submission flows through these layers before delivery to the bank.

L0

Prompt Intelligence

Parse, sanitize and risk-score user prompts. Strips celebrities, brands, political and religious references before generation.

  • LLM prompt parser
  • Restricted-keyword sanitizer
  • Abuse detection (3 strikes / session)
L1

Upload Guardrails

Validate file integrity, resolution, DPI, blur, noise. Lightweight NSFW pre-screen plus steganography checks.

  • ≥ 1713×1080, 600 DPI
  • Laplacian blur detection
  • LSB steganography scan
L2

Image Analysis

Multi-model vision: NSFW, faces, celebrity match, logo/trademark, OCR, object detection, CLIP concept similarity.

  • NudeNet, InsightFace, YOLOv8
  • PaddleOCR for text extraction
  • OpenCLIP concept embeddings
L3

Risk Scoring Engine

Weighted aggregation of all upstream signals into a unified risk score. Hard blockers override the formula.

  • Weighted multi-signal model
  • Hard-block list (weapons, CSAM, hate)
  • Cohort-aware thresholds
L4

Smart Auto Approval

Routes safe submissions straight to embosser delivery. Targets >80% automation.

  • Auto / Quick Review / Manual / Reject
  • Cohort-specific approval rates
  • Time-bounded quick-review SLA
L5

Fraud Detection

Behavioral signals + perceptual hashing to catch bypass attempts and abusive accounts.

  • pHash duplicate detection
  • Submission velocity monitoring
  • Per-user risk scoring
L6

Continuous Learning

Closes the loop. Ops decisions feed a labeled dataset that retrains weights and detectors over time.

  • Decision feedback collection
  • Risk weight tuning
  • Read-only for first 3–6 months
